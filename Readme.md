# TaskPad Backend

Backend API for the **TaskPad** website.

Frontend repository: [Raghav1825/TaskPad](https://github.com/Raghav1825/TaskPad)

## About

This backend is built with **Node.js**, **Express.js**, and **ES Modules**. It is intended to provide APIs for the TaskPad frontend, including task/kanban data handling, middleware, routes, controllers, and database logic.

## Tech Stack

- Node.js
- Express.js
- MongoDB
